# Privacy Policy English

This extension (Xiaohongshu Comment Export to CSV Tool) respects and protects users' privacy. This policy explains how data is handled by the extension.

1. Data Collection  
The extension only extracts publicly visible comments and basic user information (such as user ID, nickname, gender, age, constellation, profile description, comment content, likes, and reply counts) from Xiaohongshu pages when the user initiates the operation.

2. Data Storage and Transmission  
All extracted data is processed and stored locally on the user's device. No data is uploaded or transmitted to the developer or any third party.

3. Third-Party Services  
This extension does not integrate any third-party services, and does not contain advertisements or tracking codes.

4. User Responsibility  
The extension is provided solely as a technical tool. Users are responsible for ensuring their use complies with Xiaohongshu's terms of service and applicable laws.

5. Contact Information  
For any inquiries, please contact the developer: 941521358@qq.com
